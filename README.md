# fuctibash
A shell script for Arch Linux usability. Anyone can contribute to this repo.

# Installing
## Let' s clone repo!
`git clone https://github.com/Quitaxd/fuctibash.git`

## Enter directory
`cd fuctibash`

## Execute script
`./fuctibash.sh`

NOTE: If you getting a error type `chmod +x fuctibash.sh` and retry.
